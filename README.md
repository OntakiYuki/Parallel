**平行テーマ**はSCP-CNウィキ用に開発されたCSSテーマです。Dr HormressがBasaltテーマおよび**Bedrockテーマ**を大幅に改良し、二次開発を容易にするために設計しており、磨きのかかった外観および複数のテーマプリセットが提供されています。当初、**平行テーマ**は直接利用されることを想定して作られたものではありませんでしたが、今では誰でも使うことができます。

このリポジトリは、開発中の**平行テーマ**のコード統合バージョンです。このバージョンが完成すれば、平行テーマは**Bedrockテーマ**のコードへの直接参照に依存しなくなります。
